**最新修復與重構（2026-05-03）：**

**修復 #1（chainKills）：**
- 問題：主角只攻擊一次就停止，子彈打不到怪物
- 原因：`game.js:448` 引用未定義變數，ReferenceError 打斷遊戲循環
- 解決：移動變數定義至使用前
- Commit: 6350085

**修復 #2（敵人無法被命中）：**
- 問題：子彈射出後無法命中敵人（敵人血量不下降）
- 原因：`enemyGrid` 未更新，`getNearby()` 返回空陣列，碰撞檢測無效
- 解決：每幀 `clear()` + `insert()` 所有敵人到 SpatialGrid
- Commit: f5926c7

**修復 #3（主角射擊停止）：**
- 問題：主角射擊一次後就停止射擊
- 原因：`player.update()` 未被調用，`fireCooldown` 不減少，`canFire()` 永遠返回 false
- 解決：在 `game.update()` 中調用 `player.update(dt, keys, canvasWidth, canvasHeight)`
- Commit: 8b79a6e

**Phase 1 重構（Update Loop + 調試工具）：**
